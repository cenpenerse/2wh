### [프로젝트 구조도]

/ (루트 경로)
├── index.jsp (메인 페이지)
│   - 역할: 사이트 첫 화면, 펫시터 검색 기능 제공.
│   - 연결: header.jsp, footer.jsp 포함. sitter.jsp로 폼 데이터 전송.
│
├── login.jsp (로그인)
│   - 역할: 로그인 폼 제공 및 사용자 인증 처리.
│   - 연결: 성공 시 index.jsp로 이동, 실패 시 에러 메시지 표시.
│
├── join.jsp (펫시터 가입 약관 동의)
│   - 역할: 펫시터로 가입하려는 사용자의 약관 동의 페이지.
│   - 연결: 동의 후 member/register.jsp로 이동.
│
├── header.jsp, footer.jsp (공통 레이아웃)
│   - 역할: 모든 페이지 상단/하단에 포함되는 공통 메뉴 및 정보.
│
├── member/ (회원 관련 기능)
│   ├── register.jsp (회원가입)
│   │   - 역할: 일반 회원 및 펫시터 가입 폼 제공 및 처리.
│   │   - 연결: 성공 시 login.jsp로 이동.
│   │
│   ├── mypage_form.jsp (마이페이지)
│   │   - 역할: 로그인한 사용자의 정보 표시.
│   │   - 연결: member_updateform.jsp, member_deleteform.jsp로 연결.
│   │
│   ├── member_updateform.jsp (회원정보 수정 폼)
│   │   - 역할: 닉네임, 비밀번호 등 개인정보 수정 폼.
│   │   - 연결: member_update.jsp로 폼 데이터 전송.
│   │
│   ├── member_update.jsp (회원정보 수정 처리)
│   │   - 역할: 수정된 정보를 DB에 반영.
│   │   - 연결: 성공 시 mypage_form.jsp로 이동.
│   │
│   ├── admin_member_list.jsp (관리자용 회원 목록)
│   │   - 역할: 전체 회원 목록 조회 및 관리.
│   │   - 연결: admin_member_delete.jsp로 삭제 요청.
│   │
│   └── ... (member_deleteform.jsp, member_delete.jsp, admin_member_delete.jsp 등)
│
└── board/ (게시판 관련 기능)
    ├── board.jsp (게시판 선택)
    │   - 역할: 이용후기/자유게시판 선택 메뉴.
    │   - 연결: board_list.jsp, free_board_list.jsp로 연결.
    │
    ├── board_list.jsp (이용후기 목록)
    │   - 역할: 페이징, 검색 기능이 포함된 게시글 목록 표시.
    │   - 연결: board_detail.jsp(상세보기), board_writeform.jsp(글쓰기)로 연결.
    │
    ├── free_board_list.jsp (자유게시판 목록)
    │   - 역할: board_list.jsp와 동일한 기능의 자유게시판.
    │
    ├── board_detail.jsp (게시글 상세보기 - Controller)
    │   - 역할: 데이터 조회, 조회수 증가 등 로직 처리.
    │   - 연결: 처리 후 board_detailform.jsp로 포워딩.
    │
    ├── board_detailform.jsp (게시글 상세보기 - View)
    │   - 역할: Controller가 전달한 데이터를 화면에 표시.
    │   - 연결: board_updateform.jsp(수정), board_delete.jsp(삭제)로 연결.
    │
    ├── board_writeform.jsp (글쓰기 폼)
    │   - 역할: 새 게시글 작성 폼.
    │   - 연결: board_write.jsp로 폼 데이터 전송.
    │
    ├── board_write.jsp (글쓰기 처리)
    │   - 역할: 작성된 게시글을 DB에 저장.
    │   - 연결: 성공 시 board_list.jsp로 이동.
    │
    ├── board_updateform.jsp (글 수정 폼)
    │   - 역할: 기존 게시글 수정 폼.
    │   - 연결: board_update.jsp로 폼 데이터 전송.
    │
    └── board_update.jsp (글 수정 처리)
        - 역할: 수정된 게시글을 DB에 반영.
        - 연결: 성공 시 board_detail.jsp로 이동.

/java/ (자바 클래스)
├── DTO (Data Transfer Object) - 데이터 객체
│   ├── member.java: 회원 정보
│   ├── board_email_post.java: 게시글 정보
│   └── board_email_comment.java: 댓글 정보
│
├── DAO (Data Access Object) - 데이터베이스 처리
│   ├── memberDBDao.java: 회원 관련 DB 연동
│   ├── board_email_postDBDao.java: 게시글 관련 DB 연동
│   └── board_email_commentDBDao.java: 댓글 관련 DB 연동
│
└── Util (유틸리티 클래스)
    └── Pagination.java: 페이징 계산 로직
